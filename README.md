# Simplestrap Theme
기존 XE1에 있던 simplestrap을 XE3로 포팅한 플러그인

## 제외된 기능
- 일반 - 프레임 고정 모듈
- 상단바 - 소셜 XE
- 상단바 - 회원 포인트
- 사이드바 - 사용자 지정 위젯 1 ~ 5
- 커스텀 / 고급 - CSS3PIE
- 커스텀 / 고급 - XE CSS 제거
- 커스텀 / 고급 - XE Validator Message
- 커스텀 / 고급 - custom_setting
- 커스텀 / 고급 - 라이센스 제거
- 점보트론 - 점보트론 관련 내용
- 슬라이드 - 슬라이드 사용 ('예', '아니요'만 사용)
- 슬라이드 - 슬라이드 표시 모듈
